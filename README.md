# Inverted-Search

📌 Overview
Inverted Search is a C-based project that creates an indexed database for fast and efficient word searching across multiple text files. Instead of scanning every file during each search operation, the program stores words along with their file references using hashing and linked list techniques.

This project demonstrates the implementation of data structures, file handling, hashing, and dynamic memory management concepts commonly used in search engines and document retrieval systems.

🚀 Features
- Create database from multiple text files
- Efficient word searching using hashing
- Display database contents
- Update existing database dynamically
- Save database into a file
- Load database from saved file
- Handle duplicate words effectively
- Fast retrieval of searched data

🛠 Technologies Used
- C Programming
- Data Structures
- Hashing
- Single Linked List
- File Handling
- Dynamic Memory Allocation

 📚 Pre-requisites
- Basics of C Programming
- Understanding of Hashing
- Knowledge of Linked Lists
- File Handling Concepts
